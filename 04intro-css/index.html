Class Selection

The Main Classes

There are three main classes to choose from to start, which also each contain subclasses. They are:

Warlock
A scholar of the D2 universe, powerful in all the magic and abilities that guardians wield
Hunter
A rogue type character and a scout, excelling at stealth and big damage output
Titan
Unbreakable walls of Destiny 2, Titans can hold the line with big melee damage and increased resistances

While each main class has an over arching identity their subclasses can further refine the way that you play.
